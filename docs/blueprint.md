# **App Name**: libly.space

## Core Features:

- Admin Authentication: Secure admin login with simple email/password authentication. One admin per library.
- Library Configuration: Configure the library with the total number of seats and the available shifts, along with the timings and capacities.
- Student Management: Add, edit, and view student profiles, tracking their status.
- Seat Allocation: Allow the admin to allocate seats to students based on available seats in different shifts.
- Payment Tracking: Manually record student payments and calculate dues and outstanding payments.
- Dashboard and Reminders: Display an admin dashboard to quickly review total strength, payment dues, total dues, seats available, and month-wise total collection.

## Style Guidelines:

- Primary color: #C6B2E3, to convey knowledge and focus, reflecting the academic setting of a library.
- Background color: #89CFF0, offering a soft contrast to make content easily readable.
- Accent color: #89CFF0, used for interactive elements and calls to action, providing a vibrant contrast against the primary color.
- Body and headline font: 'Poppins' and 'DM Sans', humanist sans-serif with a balance of modern appeal and slight warmth, ideal for maintaining readability and a friendly user interface.
- Code font: 'Source Code Pro' for displaying any code snippets or technical information within the application.
- Use flat, minimalist icons to represent different sections and actions, ensuring clarity and ease of use.
- Implement subtle transitions and animations for a smoother user experience when navigating and interacting with elements.
